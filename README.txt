Overview:

This program reads a CNF (Conjunctive Normal Form) instance and finds all satisfying assignments for the given boolean variables. It is designed to solve MAXSAT problems by checking all possible combinations of truth values for the variables.

------------------------------------------------------------------------------

Requirements:

- Python 3.x installed on your machine.

- No external libraries are required.

------------------------------------------------------------------------------

Execution:

Run the following instruction on a terminal where the lab1.py is located:

-> python lab1.py

Afterwards, it'll prompt to enter the file name. Here it refers to the name of the CNF file, for example, the one included, hoos.cnf

The program will then print all the possible solutions that satisfy the problem.
